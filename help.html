Search Parameters

User can define the following parameters when search. Each parameter has a default and some restrictions.

Title

Return only issues in series whose title matches the input.
Format: Text input box.
Restriction: Cannot be empty if checked "search by title".

Initial letter

Return only issues in series whose title starts with the input.
Format: Text input box.
Restriction: Cannot be empty if checked "search by initial letter".

Series start year

Return only issues in series whose start year matches the input.
Format: Selection menu.
Default: All.
Options: Every year from 1939 (when Marvel was founded) to today.

Issue format

Filter by the issue format.
Format: Selection menu.
Default: All.
Options: Comic, Magazine, Trade paperback, Hardcover, Digest, Graphic novel, Digital comic, Infinite comic.

Order by

Order the result set by a field.
Format: Selection menu.
Default: Default order returned by the server.
Options: Title, Issue number.

Search Result

The search will return the first Marvel character (in alphabetical order) that appeared in each comic satisfying the request. Each search will return at most 100 results. Each page shows at most 10 results. Information returned include:

Name

The name of the character.

Thumbnail

The thumbnail imgae of the character if there is one available.

Title

Title of the comic that satisfies the request and the character appears in.

Description

Description of the character if there is one available.
